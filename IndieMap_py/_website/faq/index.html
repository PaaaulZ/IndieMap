  • What is this?

    This is a map of all the cities named in lyrics of italian indie songs. For some weird reason italian indie artists love naming songs after cities or just throwing random cities in their lyrics so I decided to write a simple script to put everything into a map.

  • Why?

    Why not? At the start it was all a meme, now it's "statistics".

  • How does it work?

    Just click a point on the map to see the name of the city and a list of songs. Once you click on a song the lyrics will pop on the right and you can read where in the lyrics the song is named by the artist.

  • How do you choose songs?

    My goal was to make the data gathering almost 100% automatic so I don't choose songs. Every time you run the scripts it gathers a list of artists, scans every song they have and if it finds a city name in the lyrics it will save the coordinates just to later place everything on the map.

  • Where can I learn more?

    You can read more detailed informations and code on my GitHub project page by clicking here IndieMap @ GitHub. Please do not expect advanced stuff, this is just for fun and games.
